# Plan_v0.2 — 텍스트 서식 툴바

작성일: 2026-03-31

## 목표

에디터에 Apple Notes 스타일의 서식 입력 툴바를 추가한다.
마크다운 문법을 직접 타이핑하지 않아도 버튼 클릭으로 서식을 적용할 수 있게 한다.

---

## 완료된 작업

### 신규 파일
- `lib/widgets/formatting_toolbar.dart` — 서식 툴바 위젯

### 수정된 파일
- `lib/widgets/note_editor.dart` — 데스크탑/macOS/Windows 에디터에 툴바 삽입
- `lib/screens/mobile_editor_screen.dart` — 모바일 에디터에 툴바 삽입

---

## 툴바 구성

```
[B] [I] [S] | [Aa▼] | [표]
```

| 버튼 | 기능 | 마크다운 변환 |
|------|------|--------------|
| B | 굵게 | `**텍스트**` |
| I | 기울임 | `*텍스트*` |
| S | 취소선 | `~~텍스트~~` |
| Aa▼ | 문단 스타일 팝업 | 아래 참조 |
| 표 | 표 삽입 | 마크다운 표 템플릿 |

### 문단 스타일 팝업 (Aa▼)

| 항목 | 마크다운 |
|------|---------|
| 제목 | `# ` |
| 머리말 | `## ` |
| 부머리말 | `### ` |
| 본문 | 접두어 제거 |
| 모노 스타일 | `` `텍스트` `` |
| • 구분점 목록 | `- ` |
| – 대시 목록 | `– ` |
| 1. 번호 목록 | `1. ` |
| \| 블록 인용 | `> ` |

---

## 동작 방식

- **인라인 서식** (B, I, S, 모노): 선택된 텍스트를 마크다운 기호로 감쌈. 선택 없으면 커서 위치에 삽입.
- **줄 접두어** (제목, 목록 등): 현재 줄 맨 앞에 접두어 삽입. 기존 접두어는 자동 제거 후 교체.
- **표 삽입**: 커서 위치에 3열 2행 마크다운 표 템플릿 삽입.
- **표시 조건**: 편집 모드에서만 보임. 미리보기 모드 전환 시 자동 숨김.

---

## 검증 방법

1. `flutter run -d macos` 실행
2. 노트 선택 → 편집 모드 진입
3. 텍스트 선택 후 B/I/S 버튼 클릭 → 마크다운 기호 삽입 확인
4. Aa▼ 클릭 → 팝업 열림 → 스타일 선택 → 줄 앞에 접두어 삽입 확인
5. 표 버튼 클릭 → 표 템플릿 삽입 확인
6. 미리보기 모드 → 툴바 숨김 확인 + 서식 렌더링 확인
7. `flutter run -d windows`에서 동일 확인
