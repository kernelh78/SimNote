# SimNote 개발 계획 v0.1

> 클라우드 없이, 내 기기끼리만 동기화되는 개인 메모 앱

작성일: 2026-03-28

---

## 한 줄 목표

**Wi-Fi 안에서 내 폰과 PC가 알아서 동기화되는 메모 앱. 데이터는 내 기기 밖으로 절대 안 나간다.**

---

## 기술 선택

| 항목 | 선택 | 이유 |
|------|------|------|
| 개발 언어/프레임워크 | Flutter | 안드로이드/iOS/Windows/macOS 하나의 코드로 대응 |
| 로컬 데이터베이스 | Isar | 빠르고, 해시태그 인덱싱에 유리, Flutter 전용 최적화 |
| 기기 탐색 | mDNS | 같은 Wi-Fi 안에서 앱 설치된 기기 자동 발견 (에어드랍 방식) |
| 데이터 전송 | TCP 소켓 | 안정적인 전송 보장 |
| 암호화 | AES-256 | 전송 데이터 보호 |

---

## 데이터 구조

메모 하나가 어떻게 저장되는지:

```
Notebook (폴더)
└── Note (메모)
    ├── 제목
    ├── 본문
    ├── 생성일
    ├── 수정일 ← 동기화 충돌 해결에 핵심
    ├── 즐겨찾기 여부
    └── Tags (해시태그 목록)
```

---

## 화면 구성

- **사이드바**: 폴더 목록 / 즐겨찾기 / 태그 목록
- **메인 영역**: 선택한 폴더의 메모 리스트 + 편집창
- **동기화 패널**: 같은 Wi-Fi 안에서 발견된 내 기기 목록, 마지막 동기화 시각

---

## 개발 단계

### 1단계 — 혼자 쓰는 메모장 만들기
**목표:** 앱 설치하면 바로 메모 쓸 수 있는 상태

- [ ] Flutter 프로젝트 생성 (Android + iOS + Windows + macOS 대상)
- [ ] Isar DB 연동 — Notebook, Note, Tag 테이블 설계
- [ ] 메모 작성 / 수정 / 삭제
- [ ] 폴더(Notebook) 생성 및 메모 분류
- [ ] 해시태그 추가 및 태그별 필터링
- [ ] 즐겨찾기 기능
- [ ] 실시간 검색 (제목 + 본문 + 태그)

**완료 기준:** 폰 또는 PC 단독으로 메모 앱으로서 완전히 작동한다

---

### 2단계 — 같은 Wi-Fi 안의 기기 찾기
**목표:** 버튼 하나로 내 다른 기기가 목록에 뜬다

- [ ] mDNS로 같은 네트워크의 SimNote 설치 기기 자동 탐색
- [ ] 기기 이름 / IP 주소 표시
- [ ] 탐색 결과를 동기화 패널에 실시간 업데이트

**완료 기준:** 폰과 PC가 같은 Wi-Fi에 있으면 서로를 자동으로 발견한다

---

### 3단계 — 기기 간 데이터 주고받기
**목표:** 두 기기가 메모 데이터를 실제로 교환한다

- [ ] 최초 연결 시 PIN 번호 입력으로 기기 인증
- [ ] 인증된 기기 목록 저장 (다음부터는 PIN 불필요)
- [ ] TCP 소켓으로 메모 데이터(JSON) 송수신
- [ ] 전송 성공/실패 상태 표시

**완료 기준:** 폰에서 쓴 메모가 버튼 하나로 PC에 나타난다

---

### 4단계 — 충돌 없는 똑똑한 동기화
**목표:** 같은 메모를 두 기기에서 동시에 수정해도 데이터가 안 날아간다

- [ ] 수정일(lastModified) 타임스탬프 비교로 최신 버전 판별
- [ ] 충돌 발생 시 사용자에게 선택권 제공 (어떤 버전을 유지할지)
- [ ] 삭제된 메모 처리 (한쪽에서 삭제 + 다른 쪽에서 수정 케이스)
- [ ] 동기화 로그 — 뭐가 어떻게 바뀌었는지 확인 가능

**완료 기준:** 동시 수정 상황에서도 데이터 손실이 없다

---

### 5단계 — 보안 강화
**목표:** 같은 Wi-Fi 안에 다른 사람이 있어도 내 메모는 안전하다

- [ ] 전송 데이터 AES-256 암호화
- [ ] 기기 고유 ID 기반 인증 (승인된 기기만 동기화 가능)
- [ ] 비승인 기기의 접근 시도 차단 및 알림

**완료 기준:** 네트워크 패킷을 가로채도 내용을 읽을 수 없다

---

## 지금 당장 시작할 것

**1단계부터.** 동기화보다 메모 자체가 먼저야.

1단계가 완성되어야 동기화할 데이터가 생긴다.

---

## 주의할 것 (나중에 반드시 부딪힐 문제)

**동기화 충돌** — 폰과 PC에서 동시에 같은 메모를 수정하면?
→ 4단계에서 해결. 지금은 타임스탬프 비교로 단순하게 처리하고, 필요하면 CRDT 방식으로 업그레이드.
